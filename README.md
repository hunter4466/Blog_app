## Getting Started

### Installation

1. Clone the repo

   git clone https://github.com/hunter4466/Blog_app.git
  

### Usage

    ```sh
    Execute "$ bundle install" command
    ```

### Test

For testing the classes run:

   ```sh
   bundle exec rspec spec
   ```
